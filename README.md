# 🤖 Previsão de Score de Crédito com Machine Learning

## 📌 Sobre o projeto

Este projeto simula um cenário real de negócio em uma instituição financeira fictícia, onde o objetivo é prever o score de crédito de clientes com base em dados históricos.

A proposta é utilizar técnicas de Machine Learning para auxiliar a tomada de decisão na concessão de crédito, identificação de riscos e definição de estratégias financeiras.

---

## 🎯 Problema de negócio

Instituições financeiras precisam avaliar o risco de inadimplência antes de conceder crédito.

Neste contexto, o projeto busca desenvolver um modelo capaz de classificar clientes de acordo com seu score de crédito, contribuindo para decisões mais seguras e eficientes.

---

## 📊 Base de dados

A base contém informações como:

- idade  
- profissão  
- renda anual  
- histórico de crédito  
- número de contas e cartões  
- empréstimos ativos  
- taxa de juros  
- comportamento de pagamento  
- score de crédito (variável alvo)

---

## 🛠️ Etapas realizadas

- Importação e exploração da base de dados  
- Tratamento de variáveis categóricas  
- Preparação das variáveis para modelagem  
- Separação entre dados de treino e teste  
- Treinamento de modelos de classificação  
- Avaliação de desempenho por meio da acurácia  
- Identificação das variáveis mais relevantes para previsão  

---

## 🤖 Modelos utilizados

- Árvore de decisão  
- K-Nearest Neighbors (KNN)

O modelo de árvore de decisão apresentou melhor desempenho no conjunto analisado.

---

## 📈 Resultados

O modelo selecionado atingiu aproximadamente **82% de acurácia**, demonstrando boa capacidade preditiva para classificação do score de crédito dos clientes.

A análise também permitiu identificar variáveis importantes para definição do score, contribuindo para entendimento do comportamento financeiro dos clientes.
