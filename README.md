# Credit Score Prediction with Machine Learning

Machine Learning project that predicts customer credit score using classification models in Python.

The objective of this project is to simulate a real-world banking scenario where a financial institution needs to evaluate whether a customer is a good candidate for credit based on historical data.

The project includes data preprocessing, feature transformation, model training, evaluation, and analysis of the most important variables influencing credit score.

---

## Project Overview

In the banking sector, credit score is an important metric used to determine whether a customer is eligible for loans or credit products.

In this project, we analyze a dataset containing information about bank clients and build classification models capable of predicting the customer's credit score.

The workflow includes:

- Importing and exploring the dataset
- Handling and transforming categorical variables
- Splitting data into training and testing sets
- Training multiple classification models
- Comparing model performance
- Identifying the most important features for prediction

---

## Dataset

The dataset contains information about **100,000 customers and 25 variables**, including financial and behavioral characteristics.

Examples of features:

- Age
- Profession
- Annual salary
- Credit cards
- Loans
- Payment behavior
- Credit mix
- Total debt
- Interest rate

The target variable is:

**score_credit**

Which represents the credit classification of the client.

Possible categories include:

- Poor
- Standard
- Good
